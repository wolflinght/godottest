# 《坛说：江湖》角色系统详细策划文档 (Data Foundation v1.0)

## 1. 角色核心数据结构概述

角色系统分为四大核心模块，所有模块的数据汇总构成玩家的完整档案（Profile）：

- **基础档案与状态** (Identity & State)
- **属性面板** (Attributes)
- **武学档案** (Martial Arts & Loadout)
- **行囊与装备** (Inventory & Equipment)

---

## 2. 基础档案与生活状态 (Identity & State)

| 字段 | 说明 |
|------|------|
| 门派 (Sect) | 决定可拜师的NPC、门派专属任务、善恶限制（少林、武当、血刀、散人） |
| 年龄 (Age) | 建号默认14岁。每挂机48小时长1岁。某些武学随年龄增长威力变大；60岁后若未突破内功境界，基础属性衰退 |
| 潜能 (Potential) | 打怪/做任务获得，消耗用于提升武学等级 |
| 阅历 (Experience) | 不提升等级，仅用于解锁高级地图和高阶武学的学习前置条件 |
| 饱食度 / 饮水度 | 上限100，随时间/战斗消耗。低于20无法打坐/练功；降为0气血上限临时减半。消耗【包子】【清水】恢复 |
| 外貌 (Appearance) | 建号时由"根骨+福缘"生成（宛如天仙/相貌平平/面目可憎）。频繁重伤会留疤痕导致外貌下降，影响部分NPC交互态度 |

---

## 3. 属性面板系统 (Attributes System)

### 3.1 四大先天属性 (Base Stats)

建号时Roll点生成，**总和固定80点**，单项最高30，最低10。后期极难改变，决定培养流派。

| 属性 | 核心作用 | 对应流派偏好 |
|------|----------|-------------|
| 臂力 (STR) | 物理攻击力、招架概率、负重上限 | 刀法、重剑、外家拳法（降龙十八掌） |
| 身法 (AGI) | 战斗集气速度（行动频率）、闪避率 | 剑法、暗器、轻功流（辟邪剑法） |
| 根骨 (CON) | 初始气血上限、气血成长率、内功防御力 | 挨打反伤流、高血量容错流 |
| 悟性 (INT) | 降低升级武学消耗的潜能、触发奇遇/绝学概率 | 散人（学百家武学）、法系（依赖大招） |

### 3.2 隐藏属性 (Hidden Stats)

不在主面板显示数值，通过文字描述暗示（"你看起来一身正气" / "你最近似乎霉运缠身"）。

| 属性 | 说明 |
|------|------|
| 福缘 (Luck) | 影响暴击率、打怪掉宝率、触发隐藏奇遇概率 |
| 善恶 (Morality) | 范围 -1000 ~ 1000。影响正邪路线、衙门通缉状态 |

### 3.3 战斗面板属性 (Combat Stats)

由**基础属性 + 装备 + 武学等级**共同计算，直接参与战斗底层公式：

- 气血 (HP) 与 内力 (MP)
- 攻击力 (ATK) 与 防御力 (DEF)
- 集气速度 (SPD) 与 闪避/命中/招架率

## 4. 武学档案与配置系统 (Martial Arts Loadout)

### 4.1 已掌握武学池 (Learned Skills)

- 记录通过拜师、奇遇学到的所有武功及等级
- **等级上限**受限于【基本功】（如《太极剑》等级不能超过《基本剑法》）
- **属性加成**：武功每升1级提供微量面板属性（如《太极剑》每10级 +1点身法）

### 4.2 战斗预设槽位 (Combat Loadout)

紧密联动战斗文档V1.3的智能切换系统。必须在**非战斗状态**下装配：

| 槽位 | 说明 |
|------|------|
| 主修内功 (Neigong) | 决定内力上限、战斗被动增益（如九阳神功回血）、吸气回血量 |
| 主修轻功 (Qinggong) | 决定大地图移动速度、战斗基础闪避率 |
| 【拳脚】预设外功 | 空手时默认使用的功法（如：太极拳） |
| 【刀法】预设外功 | 装备刀类时默认使用的功法（如：八卦刀） |
| 【剑法】预设外功 | 装备剑类时默认使用的功法（如：太极剑） |

> 注：某类兵刃未设置预设功法时，战斗中拿该兵刃点击攻击将触发【乱挥惩罚】。

---

## 5. 行囊与装备系统 (Inventory & Equipment)

### 5.1 装备槽位 (Equipment Slots)

极简文字流，保留4个核心槽位：

| 槽位 | 说明 |
|------|------|
| 兵刃 (Weapon) | Tag：拳套/刀/剑。决定攻击力。战斗中可通过[换兵刃]切换 |
| 衣服 (Armor) | 决定防御力和最大气血 |
| 鞋履 (Shoes) | 决定额外闪避和集气速度 |
| 饰品 (Accessory) | 特殊词条（如防点穴、福缘+5） |

### 5.2 行囊系统 (Inventory)

**负重机制**：每个物品都有重量，最大负重由【臂力】决定。超重无法移动（提示：你身上背的东西太重了，迈不开腿。）

**物品分类：**

| 类型 | 示例 |
|------|------|
| 消耗品 | 金疮药（战外回血）、包子（恢复饱食度）、叫花鸡（可能触发奇遇） |
| 材料 | 铁矿、草药（用于生活职业打造） |
| 任务/书信 | 顾炎武的信、神秘的羊皮卷轴 |
